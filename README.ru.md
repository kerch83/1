# Децентрализованная P2P-платформа для общения и сообществ

Это проект с открытым исходным кодом для создания децентрализованной, peer-to-peer (P2P) платформы для общения. Наша цель — создать самомодерируемую, устойчивую к цензуре сеть, в которой ценность определяется вкладом сообщества, а не финансовой мощью.

## Ключевые концепции

*   **P2P-сеть**: Платформа работает на основе peer-to-peer сети, использующей технологию WebRTC. Центрального сервера для обработки сообщений нет.
*   **Блокчейн-комнаты**: Каждая чат-комната представляет собой отдельный, независимый блокчейн.
*   **"Время" как валюта**: Внутренняя экономическая система, основанная на "Времени" и "Репутации", для вознаграждения ценного вклада.
*   **Лицензия AGPLv3**: Проект распространяется под лицензией GNU AGPLv3, чтобы гарантировать, что он останется открытым и свободным.

## Технологический стек

*   **Монорепозиторий**: Управляется с помощью npm workspaces.
*   **Веб-приложение**: Vue.js + Ionic
*   **Бот**: Node.js + Telegraf
*   **База данных**: RxDB (с локальным приоритетом и P2P-репликацией)
*   **Общая логика**: Ядро блокчейна и экономическая логика, используемые совместно веб-приложением и ботом.

## Структура проекта

*   `/packages/webapp`: Фронтенд на Vue/Ionic.
*   `/packages/bot`: Telegram-бот, выступающий в роли узла сети.
*   `/packages/shared`: Общая логика, схемы и утилиты.
