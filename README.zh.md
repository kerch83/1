# 去中心化P2P聊天与社区平台

这是一个开源项目，旨在构建一个去中心化的点对点（P2P）通信平台。我们的目标是创建一个自我管理、抗审查的网络，其价值由社区贡献决定，而非财力。

## 核心概念

*   **P2P网络**: 该平台基于WebRTC的点对点网络运行。没有用于消息传递的中央服务器。
*   **区块链房间**: 每个聊天室都是一个独立的区块链。
*   **“时间”作为货币**: 一个基于“时间”和“声誉”的内部经济系统，用于奖励有价值的贡献。
*   **AGPLv3许可证**: 该项目根据GNU AGPLv3许可证授权，以确保其保持开放和免费。

## 技术栈

*   **Monorepo**: 使用npm workspaces管理。
*   **Web应用**: Vue.js + Ionic
*   **机器人**: Node.js + Telegraf
*   **数据库**: RxDB (本地优先，P2P复制)
*   **共享逻辑**: Web应用和机器人共享的核心区块链和经济逻辑。

## 项目结构

*   `/packages/webapp`: Vue/Ionic前端。
*   `/packages/bot`: 作为网络节点的Telegram机器人。
*   `/packages/shared`: 共享逻辑、模式和实用程序。
